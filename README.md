# nsgdown
Script to download node-style-guide files

## Install

### Ubuntu
```
sudo curl https://raw.githubusercontent.com/jonatasleon/nsgdown/master/nsgdown.sh -o /usr/local/bin/nsgdown && sudo chmod 731 /usr/local/bin/nsgdown
```

### As root
```
sudo curl https://raw.githubusercontent.com/jonatasleon/nsgdown/master/nsgdown.sh -o /usr/local/bin/nsgdown && sudo chmod 731 /usr/local/bin/nsgdown
```
